1.  **Clone the repository:**

    ```bash
    git clone https://github.com/AshwiniParaye1/nullBrains.git
    ```

2.  **Navigate to the project directory:**

    ```bash
    cd nullBrains
    ```

3.  **Install dependencies:**

    ```bash
    npm install
    # or
    yarn install
    ```

4.  **Start the development server:**

